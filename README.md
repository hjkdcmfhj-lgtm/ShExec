ShExec - Android Shell脚本可视化执行工具
 
一款纯原生Android开发的轻量级离线Shell脚本管理与执行工具，告别繁琐命令行操作，实现本地.sh脚本一键管控、运行与日志实时查看，为Android端脚本运行提供极简、高效的解决方案。


📢 项目简介
 
ShExec是专为Android设备量身打造的原生Shell脚本执行管理工具，无冗余第三方依赖、无内嵌广告、纯本地离线运行，全程无需联网。项目摒弃传统Termux复杂的命令行操作模式，以简洁的图形化界面，实现本地.sh脚本自动扫描、可视化管理、一键执行，同时实时同步脚本运行日志与报错信息，既满足普通用户快速运行脚本的需求，也方便开发者进行脚本调试与二次开发。
 
本项目开源完整Android原生工程源码，工程结构标准规范，支持Android Studio（电脑端）、AndroidIDE（手机端）直接导入编译，无需额外配置环境，可自由进行功能拓展、定制化修改与学习参考。

 
 
✨ 核心功能特性
 
📂 本地脚本智能管理
 
自动扫描Android设备本地✈️TG下载目录中的所有.sh格式Shell脚本文件，无需手动导入、无需指定目录，快速加载并展示全部脚本；支持脚本文件直观查看，轻松区分不同脚本文件，告别命令行繁琐查找。
 
▶️ 一键脚本执行
 
摒弃传统终端命令输入模式，点击即可运行选中的.sh脚本，适配绝大多数常规Shell脚本执行逻辑，无需学习命令语法，新手也能快速上手，大幅降低脚本使用门槛。
 
📜 实时日志输出
 
脚本运行过程中，实时同步输出控制台运行日志、执行进度与报错信息，清晰查看脚本运行状态，方便快速调试脚本问题、排查执行异常，无需额外借助终端工具查看日志。
 
🚀 轻量原生流畅
 
纯Android原生代码开发，安装包体积小巧，运行流畅无卡顿、无后台驻留、无电量消耗；不依赖额外终端环境，安装即可使用，适配市面上绝大多数Android设备。
 
🛠️ 开源易拓展
 
完整原生工程源码开放，代码结构清晰、注释友好，便于开发者二次开发、新增功能、定制界面或修改执行逻辑，适合Android开发学习与实战参考。
 
🔧 编译与部署指南
 
开发环境要求
 
- 开发工具：Android Studio 4.0及以上版本 / AndroidIDE（手机端开发）
- JDK版本：JDK 8及以上
- 系统适配：Android 7.0+ 设备
- 构建工具：项目内置Gradle，无需额外下载
 
完整编译步骤
 
1. 克隆源码到本地
 
 
 
2. 导入开发工具
将项目导入Android Studio或AndroidIDE中，等待Gradle自动同步完成，期间保持网络畅通，自动下载项目所需依赖。
3. 构建APK安装包
 
- 调试版本：直接点击开发工具「运行」按钮，生成调试版APK，直接安装至设备/模拟器；
- 正式版本：点击 Build → Generate Signed Bundle/APK ，配置签名文件后，生成可正式分发的APK。
 
4. 安装运行
开启设备「未知来源应用安装权限」，安装编译生成的APK文件，即可正常使用。
 
📱 新手使用教程
 
1. 权限授予
首次打开应用，务必授予存储读取权限，否则应用无法扫描本地脚本文件；
2. 脚本准备
将需要运行的.sh格式Shell脚本，存放至Android设备任意本地存储目录；
3. 加载脚本
进入应用首页，自动扫描并加载本地所有.sh脚本，展示在主界面列表中；
4. 执行脚本
选择目标脚本，点击执行按钮，即可启动脚本运行，下方实时查看运行日志；
5. 日志查看
脚本执行完成/终止后，可回看完整日志信息，快速排查运行异常。
 
⚠️ 重要温馨提示
 
1. 部分高阶系统脚本、内核脚本，需要设备获取Root权限、或依赖Termux环境才能正常执行，普通脚本无额外环境要求；
2. 请务必使用来源合法、安全可靠的.sh脚本，避免运行恶意脚本导致设备异常、数据丢失；
3. 本项目为纯本地离线工具，不会采集、上传任何用户数据、脚本文件与运行记录；
4. 因Android设备系统定制差异、脚本语法兼容性问题，部分特殊脚本可能无法正常执行，属于正常现象。
